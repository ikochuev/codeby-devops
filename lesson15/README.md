Terraform для Yandex.Cloud: Урок 15
Проект создает инфраструктуру в Yandex.Cloud с помощью двух модулей Terraform.

Структура

lesson15/
├── main.tf           # Основная конфигурация
├── provider.tf       # Настройка провайдера
├── variables.tf      # Переменные
├── outputs.tf        # Выходные данные
├── terraform.tfvars  # Значения переменных
└── modules/          # Модули
    ├── vpc-data/     # Модуль данных о подсетях
    └── vm/           # Модуль создания ВМ
 Модули
1. Модуль vpc-data
Назначение: Получает информацию о подсетях в VPC.
Работа: Принимает ID подсетей, возвращает сгруппированные данные.

2. Модуль vm
Назначение: Создает ВМ с автоматическим выбором подсети по зоне.
Работа: Получает список подсетей, выбирает подходящую по зоне, создает ВМ.

 Запуск

# Инициализация
terraform init

# План развертывания
terraform plan

# Применение
terraform apply

# Уничтожение
terraform destroy

# Результаты
После запуска создается:
Сеть VPC
Две подсети в разных зонах
Две ВМ, каждая в своей зоне
